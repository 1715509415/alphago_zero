# alphago_zero
A reproduction of Alphago Zero in "Mastering the game of Go without human knowledge" using Tensorflow 1.3.

This code is based on 'https://github.com/Rochester-NRT/RocAlphaGo'.

Now the board size is 5.

Run train.sh 

## play
